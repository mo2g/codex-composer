## Codex App Template

- Source of truth: `docs/codex-task-card-workflow.md`, `docs/codex-debug-workflow.md`
- Skills: `planner`, `implementer`, `resume-work`, `change-check`, `debug-investigation`. Plan mode adds `task-orchestrator`.
- Default loop: plan → implement → verify → commit → merge
- Task Cards should include: verification commands, done criteria, review expectations
- Keep merge manual
