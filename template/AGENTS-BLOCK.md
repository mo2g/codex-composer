## Codex App Template

- Source of truth: `docs/codex-task-card-workflow.md` and `docs/codex-debug-workflow.md`.
- Skills: `planner`, `implementer`, `resume-work`, `change-check`, `debug-investigation`.
- Start non-trivial work with `planner`; use `docs/_codex/<task-slug>/` only when durable state helps.
- For unclear root cause, use `debug-investigation`; use `resume-work` for recovery and `change-check` before commit or manual merge.
- Keep merge manual.
