## Codex App Template

- Source of truth: `docs/codex-task-card-workflow.md` and `docs/codex-debug-workflow.md`.
- Skills: `planner`, `implementer`, `resume-work`, `change-check`, `debug-investigation`.
- Start non-trivial work with `planner`; keep durable state in `docs/_codex/<task-slug>/`.
- For unclear root cause, use `debug-investigation` and keep `debug.md` current.
- Use `resume-work` for recovery, `change-check` for the final evidence pass, and keep merge manual.
